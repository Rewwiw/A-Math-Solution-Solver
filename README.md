## 🚀 ฟีเจอร์เด่น

### ✔ โหมด Bingo 8  
- ใส่เบี้ยบนมือได้สูงสุด 9 ตัว (ตามกติกาจริง)  
- ระบบจะหา **ทุกสมการที่ถูกต้อง** ตามกติกา A-Math

### ✔ โหมด Bingo Fix Position  
- กำหนด “จำนวนตัวทั้งหมดในสมการ” ได้เอง  
- เลือกล็อกค่าบางตำแหน่งได้ เช่นใส่ 17 ไว้ที่ตำแหน่งที่ 4  
- ส่วนที่เหลือระบบจัดเรียงให้เอง  
- รองรับเลข 0–20 และเครื่องหมายทั้งหมด

### ✔ รองรับเบี้ยพิเศษ  
- `+/-` → ขยายเป็น `+` หรือ `-`  
- `x/÷` → ขยายเป็น `*` หรือ `/`  
- `?`   → ขยายได้ทั้งเลข 0–20 และ operator  
- `x` → `*`  
- `÷` → `/`

### ✔ Logic A-Math ถูกต้องตามกติกา  
- ต้องมี `=` อย่างน้อย 1 ครั้ง  
- ไม่อนุญาตเครื่องหมายซ้อนกันเช่น `+-`, `**` (ยกเว้น `=-`)  
- ไม่อนุญาตเลข 10–20 ไปติดกับตัวอื่น เช่น  
  - ❌ `7 17` → 717  
  - ❌ `10 11` → 1011  
- ไม่อนุญาต `/0`  
- ไม่อนุญาตเลขขึ้นต้นด้วย `0` เช่น `012`  
- ตรวจสมการโดยเทียบค่าซ้าย = ขวา
